# 코딩테스트 100문제 Challenge

## Project Overview

코딩테스트 대비를 위한 Javascript 기반 100문제 챌린지입니다. 매일 두시간씩 꾸준히 풀어나가는 것이 목표입니다.

## Project Structure

```
📦 project-root
├── 📁 src
│   ├── 📁 배열
│   │   ├── 📁 [문제]
│   │   │   ├── 📄 README.md
│   │   │   └── 📄 solution.js
│   │   ├── 📁 ...
│   │   └── 📄 README.md
│   ├── 📁 스택
│   │   └── 📁 ...
│   └── 📁 ...
├── 📄 package.json
└── 📄 README.md

```

- 카테고리 별로 디렉토리를 생성하여 각 문제별 하위 디텍토리 생성 & 관리 (`스택`,`큐`,`해시`,...)
- 카테고리 디렉토리에는 다음 파일들을 포함:
  - `[문제]`: 폴더
  - `README.md`: 카테고리 개념 & 보충 의견
- 각 문제별 하위 디렉토리에는 다음 파일들을 포함:
  - `README.md`: 문제 설명, 입출력 예시, 제약사항
  - `solution.js`: 솔루션 코드
