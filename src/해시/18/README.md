# 두 개의 수로 특정 값 만들기

## 문제 설명

N개의 양의 정수로 이루어진 리스트 `arr`와 정수 `target`이 주어졌을 때, 이 중에서 합이 `target`인 두 수가 `arr`에 있는지 찾고, 있으면 `true`, 없으면 `false`를 반환하는 `solution()` 함수를 작성하세요.

## 제약사항

- 배열의 길이는 2 이상 10,000 이하
- 배열의 각 원소는 1 이상 10,000 이하의 자연수
- target은 1 이상 20,000 이하의 자연수
- 같은 원소를 두 번 사용할 수 없음

## 권장 시간 복잡도 O(N+k)

## 입출력 예시

### 예시 1

```javascript
// 입력
arr = [1, 2, 3, 4, 8];
target = 6;

// 출력
true;
```

### 예시 2

```javascript
// 입력
arr = [2, 3, 5, 9];
target = 10;

// 출력
false;
```
